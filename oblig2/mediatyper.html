Media typer

Dato: 02.10.2018

Hva er en media type?

Media typer lar deg spesifisere CSS for forskjellige enheter og media typer. Det finnes flere forskjellige mediatyper som er definert i CSS, som for eksempel all, print, projection, speech, braille, tty og tv. Jeg har imidlertid valgt å kun skrive litt om all, screen og print. Dette fordi de resterende medietypene er på vei ut, siden de nå blir dekket av regler for tilgjengelighet og responsivt design.

Kort forklaring:

  • All: Tilsvarer å ikke definere en mediatype i det hele tatt, siden den vil gjelde uansett.
  • Screen: CSS-spesifikasjon for alle enheter som har en skjerm
  • Print: Brukes for å tilpasse utseende til en utskrift.

Hvordan bruker man disse?

Det finnes tre forskjellige måter å bruke mediatyper på. Første metoden er å linke til en ekstern CSS-fil:

<link rel="stylesheet" type="text/css" media="print" href="print-stilark.css/">

Man kan også bruke mediatyper i en eksisterende CSS-fil (inline), ved å lage egne @media-blokker:

@media print {
	p {
		color: black;
	}
}

Man kan også importere stilark i CSS:

@import url('print-css') print

Fordeler og ulemper

Fordelen med mediatyper er at man kan tilpasse design etter hvilken enhet som brukes. Ulempen er at man får mer, og ikke minst mer kompleks, CSS-kode.

CSS Layouts

Table

Før i tiden kunne man finne på å sette opp hele layouten til en nettside ved hjelp av et table. Da puttet man elementer inn i radene og kolonnene til tabellen, for så å style slik man ønsket. Dette er heldigvis ikke noe man gjør i dag.

CSS Float

Floating er en egen posisjoneringsteknikk i CSS. Man angir hvordan elementer skal "flyte" i forhold til hverandre. Denne ble mer brukt tidligere, før Flexbox og Grid kom på banen. Fordelen med float er den gode kompatibiliteten.

CSS Flexbox

Flexbox er en nyere, endimosjonal, posisjoneringsteknikk i CSS. Hovedideen bak flex er å gi en container mulighet til å endre dens bredde, høyde og rekkefølge. Dette for å best fylle ledig plass. Dette gjør at den også håndterer skalering av nettsiden til ulike skjermstørrelser og medietyper. En flexbeholder utvider elementer for å fylle ledig plass, eller krymper for å forhindre overflow.

CSS Grid

Grid er den nyeste, todimosjonale, posisjoneringsteknikken i CSS. Siden den er todimosjonal, håndterer den både kolonner og rader. For å bruke Grid skriver man CSS-kode både for et forelder-element, som blir containeren, og til dette elementets barn. Disse barna blir da Grid Items. Man vil altså definere grid-områder i rader og kolonner før man skriver videre kode.

Optimal løsning

Det mest optimale i dag, når det kommer til layouten på en nettside, vil være å bruke både Flexbox og Grid. Grid er mest fleksibel, mens Flexbox er den med størst vekst og har noe bedre støtte.
